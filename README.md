# bmi_calculator
This app is for calculating the BMI and the needed calories per day for men and women, and have google maps feature
to fetch the nearby places based on the user's current location

## Packages
  animated_splash_screen,
  provider,
  google_maps_flutter,
  geolocator,
  dio,
